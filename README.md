# HTML/CSS/JavaScript 학습 프로젝트 | HTML/CSS/JS Practice

## 프로젝트 소개 | About

AI 데이터 분석 풀스택 웹 개발자 양성과정(2025 3기) 
HTML, CSS, JavaScript 기초 학습 실습 프로젝트입니다.

Front-end fundamentals practice project from the AI Data Analysis Full-Stack Web Developer Training Program (2025 Cohort 3).

## 배포 | Live Demo
https://html-css-js-black.vercel.app

## 기술 스택 | Tech Stack
- HTML5
- CSS3
- JavaScript (ES6+)
- Font Awesome 6.5 (아이콘)
- Google Fonts (Nanum Gothic)

## 디렉토리 구성 | Structure

```
├── 01/   → HTML 기초
├── 02/   → CSS 기초
├── 03/   → JavaScript 기초
├── 04/   → 회문 확인
├── 05/   → 로또번호 생성
├── 06/   → 주사위
├── 07/   → 주사위 맞추기
├── 08/   → 단위 환산
├── 09/   → 랜덤수 생성 (업다운 게임)
├── 10/   → 이미지 갤러리
├── 11/   → 이미지 슬라이더
├── 12/   → 하트게임
├── img/  → 이미지 리소스
├── index.html     → 메인 페이지
├── index.css      → 메인 스타일
└── js.css         → JavaScript 관련 스타일
```

## 실행 방법 | Getting Started

별도 설치 없이 `index.html`을 브라우저에서 열면 됩니다.

```bash
git clone https://github.com/kd256k/html_css_js.git
cd html_css_js
# index.html을 브라우저에서 열기
```
